Name Type Description
options object The default plugin options.
Properties
Name Type Description
debug boolean Debug mode with console.log and in-page notification layer.
formTracking object The default options for the form tracking feature.
Properties
Name Type Description
completedEvents boolean Track the successful completion of the fields.
oneTimeOnly boolean Track the field event only the first time.
labelCompleted string The default label for the completed event.
labelNotValid string The default label for the not valid event.
labelSkipped string The default label for the skipped event.
category string The default global tracking category name.
action string The default global tracking action name.
callback callback The default callback called after tracking event.
Example
$(selector).trackMe({
      debug: false,
      formTracking {
          completedEvents: false,
          oneTimeOnly: true,
          labelCompleted: "completed",
          labelNotValid: "not valid",
          labelSkipped: "skipped"
      },
      category: "cart",
      action: "",
      callback: function() {
         console.log("Event tracked.");
      };
});
